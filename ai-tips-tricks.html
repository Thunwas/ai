BEGINNER GUIDE

AI Tricks & Tips
10 เคล็ดลับสำหรับมือใหม่

เริ่มต้นใช้งาน AI ได้อย่างมีประสิทธิภาพตั้งแต่วันแรก

01 🎯
บอก Role ให้ AI ก่อนเสมอ
กำหนดบทบาทให้ AI ก่อนถามคำถาม เพื่อให้ได้คำตอบที่ตรงจุดและมีความเชี่ยวชาญตามที่ต้องการ ไม่ใช่คำตอบแบบกว้างๆ ทั่วไป
EXAMPLE PROMPT "Act as a Senior Linux Admin. Explain how to configure…"
02 📋
ระบุ Output Format ที่ต้องการ
บอก AI ว่าอยากได้คำตอบในรูปแบบไหน เช่น Bullet, Table, Step-by-step, JSON, Markdown จะได้ผลลัพธ์ที่นำไปใช้ได้ทันที
EXAMPLE PROMPT "…ตอบเป็น Table พร้อม 3 column: ข้อดี / ข้อเสีย / Use case"
03 🔁
ใช้ Follow-up อย่าถามครั้งเดียวจบ
AI จำบริบทในการสนทนาได้ ไม่ต้องถามใหม่ทั้งหมด แค่ต่อยอดจากคำตอบก่อนหน้า เพื่อเจาะลึกได้เรื่อยๆ
FOLLOW-UP TIPS "ขยายเพิ่มในข้อที่ 3" / "ยกตัวอย่างแบบ Real-world" / "เขียนเป็น Script ให้หน่อย"
04 ✂️
ให้ Context เพียงพอ ไม่มากเกิน
ข้อมูลที่ให้ AI ต้องเพียงพอให้เข้าใจปัญหา แต่ไม่ต้องยาวเกินจำเป็น ระบุ OS, Version, Constraint ที่เกี่ยวข้องจริงๆ
STRUCTURE Context → Problem → Constraint → Expected Output
05 🧪
ขอให้ AI อธิบายก่อนทำ
โดยเฉพาะงาน Script หรือ Config อันตราย ขอให้ AI อธิบาย Logic ก่อนเสมอ เพื่อตรวจสอบความถูกต้องก่อน Run จริง
SAFETY PROMPT "อธิบายว่า Script นี้ทำอะไรทีละบรรทัด ก่อนที่ฉันจะ run"
06 🌡️
ควบคุมความ Creative ด้วยคำสั่ง
บอก AI ว่าต้องการคำตอบแบบ Conservative (ปลอดภัย) หรือ Creative (หลากหลาย) เพื่อให้ผลลัพธ์ตรงกับความต้องการ
TONE CONTROL "ตอบแบบ conservative เน้น production-safe" / "ให้ idea หลากหลาย 5 แบบ"
07 🔍
Verify คำตอบเสมอ อย่า Trust 100%
AI อาจ "Hallucinate" ข้อมูลที่ดูน่าเชื่อถือแต่ผิด โดยเฉพาะ Version, Command Syntax, API ใหม่ๆ ควรตรวจสอบกับ Official Doc เสมอ
VERIFY PROMPT "คุณแน่ใจไหมว่า command นี้ถูกต้องสำหรับ version X?"
08 🧩
แบ่งงานใหญ่เป็นชิ้นเล็กๆ
อย่าถามทุกอย่างในคำถามเดียว แบ่งงานซับซ้อนออกเป็นขั้นตอน แล้วถามทีละส่วน คุณภาพคำตอบจะดีกว่ามาก
CHUNKING STRATEGY Design → Draft → Review → Refine แทนที่จะถามทุกอย่างครั้งเดียว
09 📝
บันทึก Prompt ที่ใช้งานได้ดี
เมื่อเจอ Prompt ที่ให้ผลลัพธ์ดี ให้บันทึกเก็บไว้ใน Notion หรือ Text file สร้าง Prompt Library ของตัวเองเพื่อนำกลับมาใช้ได้เร็ว
TOOL SUGGESTION Notion / Obsidian / GitHub Gist → สร้าง Personal Prompt Library
10 🚀
ใช้ AI เป็น Thought Partner ไม่ใช่แค่ Search Engine
AI มีประโยชน์สูงสุดเมื่อใช้เป็นคู่คิด ระดมไอเดีย วิเคราะห์ Trade-off ทบทวนแผน ไม่ใช่แค่ค้นหาคำตอบเดียว
POWER PROMPT "ช่วย Devil's Advocate แผนนี้ให้หน่อย มีจุดอ่อนตรงไหนบ้าง?"
